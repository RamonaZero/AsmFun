# AsmFun
x86_64 Assembly Projects for fun

# Development Environment
- OS: Archlinux (No Xorg!)
- Architecture: x86_64 - AMD64
- Editor: Emacs
- Assembler: nasm
- Linker: gnu-ld
- Debugger: gdb (intel syntax enabled)
